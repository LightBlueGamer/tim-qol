## Features

### Bastion Automation
Automatically advances bastion turns for:
- Barracks
- Storehouse

## Documentation

Simply install the module and enable it in the settings, the module will then kick on and work it's magic!

I recommend using Calendaria which works great for custom calendars, and lengths of days.

## Notes
For use with Calendaria make sure to disable the "Advance Bastion Orders" setting as this module handles it

Currently does not work with Simple Calendar Reborn unless you keep standard time lengths (24h/day, 60m/h, 60s/m)
